# Contact & Feedback

Project ini saya buat menggunakan Laravel untuk membuat sistem **kontak dan feedback** dari pengguna.

Tujuan web ini adalah supaya pengguna bisa mengirim pesan atau saran lewat form, dan pesan tersebut akan tersimpan di database.  
Admin bisa melihat semua pesan yang masuk, dan juga bisa membalasnya langsung dari halaman admin.

## Fitur yang sudah ada
- Formulir untuk mengirim feedback
- Daftar semua feedback yang sudah dikirim
- Halaman balasan (reply) dari admin ke pengguna
- Validasi input agar data yang dikirim tidak kosong

## Rencana pengembangan
Ke depannya saya ingin menambahkan beberapa fitur seperti:
- Notifikasi kalau ada feedback baru
- Balasan otomatis ke email pengirim
- Login untuk admin
- Tampilan yang lebih menarik

## Teknologi yang digunakan
- Laravel (PHP Framework)
- MySQL untuk database
- Blade Template untuk tampilan
- Bootstrap / CSS dasar untuk styling
