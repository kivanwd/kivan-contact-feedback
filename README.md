# Contact & Feedback

Laravel Contact & Feedback Web Project

Proyek ini dibuat menggunakan Laravel untuk mengelola pesan dan feedback dari pengguna.  
Fitur utama:
- Formulir pengiriman feedback
- Tampilan daftar feedback
- Halaman balasan (reply)
- (Akan ditambahkan) notifikasi & login admin
